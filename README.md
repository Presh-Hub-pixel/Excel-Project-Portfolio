## 📊 Project Overview – Amazon Sales Dashboard 🛒

📌 **Project Description**

This project presents a comprehensive analysis of Amazon sales performance using Microsoft Excel. The dashboard provides insights into:

* Revenue performance by product category
* Regional sales distribution
* Monthly and seasonal sales trends
* Top-performing products and segments
* Key performance indicators (KPIs) for business monitoring

The objective is to convert raw transactional data into an interactive, executive-ready dashboard that supports strategic decision-making.

🔗 **View Dashboard:**
[Amazon Sales Dashboard](./Amazon%20Sales%20Dashboard/)

🔍 **Key Features**

* Dynamic Pivot Tables & Pivot Charts
* Interactive slicers and filters
* Revenue and performance tracking
* Month-on-Month trend analysis
* Power Query for data transformation
* Clean and professional dashboard layout

🎯 **Business Impact**

* Identifies top-performing product categories
* Highlights seasonal revenue trends
* Tracks key business KPIs in real time
* Supports pricing and marketing strategy decisions
